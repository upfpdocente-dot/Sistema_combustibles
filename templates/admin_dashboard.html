Información sobre Usuarios
Proceso de creación de usuarios:
  • Creación individual: Use el formulario superior para crear usuarios uno por uno
  • Creación masiva: Al cargar un archivo CSV con la columna FUNCIONARIO, los usuarios se crean automáticamente
  • Credenciales: Usuario = primera palabra del funcionario (minúsculas), Contraseña = usuario + "1234"
  • Ejemplo: Funcionario "Juan Pérez" → Usuario: "juan", Contraseña: "juan1234"
